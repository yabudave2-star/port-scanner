# Shadow's Port Scanner v1.0

A beginner-friendly port scanner built with Python and Nmap.

## Features
- Basic port scanning
- Service version detection
- Aggressive scanning
- Vulnerability scanning

## Requirements
- Python 3.x
- Nmap installed
- python-nmap library

## Installation
pip install python-nmap

## Usage
python port_scanner.py
